# Project Brief: Shear Time

Shear Time is a barber shop appointment management system designed to handle complex scheduling scenarios, specifically for hair services that require multiple time blocks (e.g., service + pause + service).

## Core Features
- **Customer Management**: Add, update, delete, and list customers.
- **Appointment Scheduling**: Book appointments for specific services or simple duration-based slots.
- **Multi-Block Services**: Support for services that span multiple time slots with defined breaks (e.g., coloring processing time).
- **Calendar Interface**: Visual representation of appointments.
- **Service Configuration**: JSON-based configuration for service types, durations, and block structures.

## Usage
The application is used by salon staff to manage their daily schedule and customer database.
